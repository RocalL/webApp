# Candidatures JEE
Projet WebServices
## Getting Started
### Prerequisites
Il faut installer : 
* Eclipse IDE for Java EE Developers
* JDK Java 8
* Git
* Tomcat 9
* Et se créer un compte Github

### Installing
Pour récupérer le projet, il faut faire la commande suivante dans le dossier de votre workspace :

```
git clone https://github.com/RocalL/webApp.git
```
